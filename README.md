## 🧠 Áreas de Experiencia y Competencias

- **Desarrollo Backend y Frontend:** Creación de APIs RESTful eficientes con Node.js y desarrollo de interfaces de usuario dinámicas, modulares y reactivas con React.
- **Programación Orientada a Objetos (POO):** Sólidos fundamentos en diseño de clases, herencia, encapsulamiento y polimorfismo aplicados a arquitecturas limpias.
- **Bases de Datos:** Modelado, normalización y optimización de consultas en sistemas relacionales (MySQL).
- **Análisis y Arquitectura de Sistemas:** Implementación de arquitectura **MVC (Modelo-Vista-Controlador)**, modelado con UML y aplicación de principios SOLID para construir software limpio, mantenible y escalable.
- **Metodologías Ágiles y DevOps:** Gestión de proyectos bajo el marco Scrum, control de versiones avanzado con Git/GitHub y contenedorización básica con Docker.
